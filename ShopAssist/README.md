# ShopAssist

A minimal Android app written in Kotlin using Jetpack Compose. It shows a single button that, when pressed, displays a list of grocery items.

## Requirements
- Android Studio (Giraffe or newer recommended)
- Android SDK Platform 34
- Gradle plugin 8.6+
- JDK 17

## How to run
1. Open this folder in Android Studio.
2. Let Gradle sync. If prompted, install missing SDKs.
3. Select a device/emulator and click Run.

The app launches with a "Show Grocery List" button. Tap it to reveal the list.
